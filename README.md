# Postify

Postify é uma aplicação para realização de operações CRUD (Criar, Ler, Atualizar e Deletar) relacionadas a posts e categorias. Nesta aplicação, os usuários podem criar, visualizar, editar e excluir posts em várias categorias.

## Funcionalidades

- Criação, leitura, atualização e exclusão de posts
- Criação, leitura, atualização e exclusão de categorias
- Relacionamento entre categorias e posts: uma categoria pode ter muitos posts

## Tecnologias Utilizadas

- Node.js
- Express.js
- PostgreSQL
- Prisma ORM
- JWT (JSON Web Tokens) para autenticação
- bcrypt para criptografia de senhas
- Outras dependências podem ser encontradas no arquivo package.json
